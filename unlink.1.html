Synopsis

unlink file

Description

The unlink command removes a directory entry, specified as file. If the directory entry cannot be removed for any reason, a diagnostic message is printed.

Operands

file

The pathname of a directory entry which is to be removed.

Diagnostics

The following error values are returned:

Error value: 0
Diagnostic: Successful completion.
Error value: >0
Diagnostic: An error occurred.

Note

On many systems, the unlink command cannot be used to remove directories.

Standards

The link utility conforms to the Single UNIX Specification, Version 3 (SUSv3), and the X/Open System Interfaces Extension (XSI).
